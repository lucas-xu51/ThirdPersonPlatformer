## 游戏规则

- 可以使用 **w**, **a**, **s**, **d** 进行方向移动
- 使用 **Space** 进行跳跃，可以实现两连跳
- 在按住方向键的基础上，点击 **e** 可以获得加速效果
- 玩家可以控制角色收获蓝色能晶
